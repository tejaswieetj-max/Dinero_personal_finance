# Dinero - Personal Finance
Financial management web app - A web programming project

## React SPA Implementation

Dinero has been successfully migrated to a React Single Page Application (SPA) with full functionality preservation.

### Features
- **Authentication**: Mock backend with login, profiles, Face ID, and PIN
- **Dashboard**: Net worth tracking and transaction history
- **Bills Management**: Pay bills with automatic balance updates
- **Goals**: Savings goals with progress tracking
- **Splits**: Debt tracking and shared expense management
- **Transactions**: Complete transaction history with filtering

### Tech Stack
- **Frontend**: React 18 + Vite
- **Routing**: React Router DOM
- **State Management**: React Context with localStorage persistence
- **Styling**: Existing CSS preserved and imported

### Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
